# mycrypt

MyCrypt é um projeto educacional de criptografia simples em C, criado para estudar leituras em stream, geração de chaves e formatos com cabeçalho. 

A implementação **não é segura** e não deve ser usada para proteger arquivos reais.  
Use apenas para fins de estudo e aprendizado.

Use por sua própria conta e risco.
